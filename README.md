# 🤖 AI Portfolio Analyzer

An AI-powered investment portfolio analysis tool built with Streamlit.

## Features

- 📊 Upload Excel trade spreadsheets
- 🤖 Automatically detects different spreadsheet headers
- 🧹 Cleans and organizes trade data
- 👥 Supports multiple clients
- 📈 Portfolio performance tracking
- 📉 Profit/loss calculations
- 🏆 Win rate analysis
- 📊 Interactive portfolio growth charts
- 💹 Yahoo Finance stock analysis
- 📅 Adjustable stock price history charts

## Portfolio Analyzer

The portfolio analyzer allows users to upload trade history files and automatically analyzes:

- Total profit/loss
- Number of trades
- Winning percentage
- Average return
- Individual stock performance
- Client portfolios

## Yahoo Finance Analyzer

The Yahoo Finance section provides:

- Company information
- Market statistics
- Stock price history
- Performance over different time periods

## Built With

- Python
- Streamlit
- Pandas
- Plotly
- Yahoo Finance API
